## Project Overview
AI Resume Analyzer is an intelligent web application that allows users to upload their resumes (PDF or DOCX) and receive an AI-based analysis. The application provides a **resume score**, identifies **strengths and weaknesses**, and gives **career improvement suggestions**. This project demonstrates the integration of **AI, FastAPI, Streamlit, and MongoDB** in a complete system.

**Unique Feature:**  
- The system analyzes resumes using AI (LLMs) to provide actionable suggestions, which is beyond a simple keyword search.
